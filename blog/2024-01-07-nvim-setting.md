## Installation ##

Install NeoVim

```
brew install neovim
```

[Install LazyVim](https://www.lazyvim.org/installation) (以 macos 為例)

```
git clone https://github.com/LazyVim/starter ~/.config/nvim
```

以上就是這麼簡單完成設定，第一次使用 nvim 會安裝預設套件，之後就可以使用了。
